English translation:
M6. No HTTP cache — Android has Cache(File(cacheDir, "http_cache"), 10*1024*1024). iOS Ktor doesn't have HttpCache plugin installed.

## What Changed
- `Han1meHttpClient.kt`: Added `install(HttpCache)` plugin

## Why Changed
- Android has HTTP caching for reducing network requests
- Ktor's HttpCache plugin provides standard HTTP caching behavior
- Respects Cache-Control headers, so Cloudflare challenges won't be cached
- Review item M6 from `review-ios-vs-android.md`

## Known Limits
- Uses in-memory cache (Ktor default). Android uses file-based cache for persistence across app restarts.
- Could add file-based cache later if needed.
