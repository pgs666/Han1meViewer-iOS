# M5: HTTP 请求重试机制

## 原始输入
M5. 没有 HTTP 重试 — OkHttp 默认 retryOnConnectionFailure(true) 对 idempotent GET 自动重试。Ktor 默认不重试。影响：弱网首页/搜索一抖就 fail。修复方向：install(HttpRequestRetry) { retryOnExceptionIf(maxRetries=2) { _, _ -> true }; exponentialDelay() }

## English Summary
Added `HttpRequestRetry` plugin to `Han1meHttpClient.kt`. Retries up to 2 times on non-DomainException exceptions (network errors, timeouts, connection failures) with exponential backoff. Does not retry on application-level errors (auth, Cloudflare, 4xx, 5xx) to avoid masking real issues.

## Changes
- `Han1meHttpClient.kt`: Added `HttpRequestRetry` plugin with exponential delay, max 2 retries, filtering out `DomainException`
